Nashik Sentinel-2 Visualization

This code utilizes the Geemap and Folium libraries to create an interactive web map focused on the Nashik region, showcasing Sentinel-2 Harmonized satellite imagery. Below are the main functionalities of the code:

Key Functionalities:

  1. Earth Engine Authentication: Authenticates and initializes the Google Earth Engine (GEE) API.
  2. Defining Area of Interest: Defines a bounding box representing Nashik's geographical boundaries using the following coordinates:

    Coordinates (with Directions):

    • Southwest Corner: (20.4255° N, 73.9398° E)
    • Northwest Corner: (20.5255° N, 73.9398° E)
    • Northeast Corner: (20.5255° N, 74.0398° E)
    • Southeast Corner: (20.4255° N, 74.0398° E)

    Coordinates of Nashik Region:

    • Nashik City: (19.9975° N, 73.7898° E)
  3. Creating an Interactive Map: Initializes a Folium map centered on Nashik with a predefined zoom level.
  4. Fetching Satellite Imagery: Fetches the Sentinel-2 Harmonized image collection based on date and location, sorted by cloud coverage.
  5. Visualization: Adds selected images to the map as tile layers with true color visualization parameters.
  6. Marker Integration: Places markers on the map for each coordinate, allowing users to see specific locations.
  7. Interactive Features: Includes search functionality and drawing tools for user interaction.
  8. Saving the Map: Saves the interactive map as an HTML file for web display.

This code serves as a powerful tool for visualizing satellite data, enabling exploration and analysis of the Nashik region through a user-friendly interface.
